# KavConsultant

Production-grade multi-tenant CRM for education consultancies.

## Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase Auth, Postgres, Storage, Realtime
- Zustand
- TanStack Query
- React Hook Form
- Zod
- Recharts
- Razorpay
- Sentry

## Run locally
1. Install dependencies.
2. Copy `.env.example` to `.env.local` and fill the values.
3. Run the Supabase migrations.
4. Start the app with `npm run dev`.

## Environment
Use the Supabase project URL and publishable key you provided in `.env.local`.
